# Deep Learning Research for Trading

This repository contains a Jupyter Notebook focused on **Deep Learning applications in trading and financial markets**, using **PyTorch** and **Scikit-learn**.

The research explores data-driven approaches to market analysis, feature engineering, model development, and performance evaluation, with an emphasis on practical experimentation rather than production deployment.

## Technologies
- Python
- PyTorch
- Scikit-learn
- Jupyter Notebook

## Research Focus
- Financial time series analysis
- Machine Learning and Deep Learning models for trading
- Model evaluation and performance analysis
